# aws-textract


## Install

Create `.env` file and below parameters value: 

JWTSECRET=<YOUR SECRET KEY>
S3ACCESSKEYID=<S3 ACCESS KEY ID>
S3SECRETKEY=<S3 SECRET KEY>
S3BUCKET=<S3 BUCKET NAME>
SESREGION=<REGION>

```console
$ npm install

Start Client 
 $ npm start
 
Start Server 
 $ npm server 

```
